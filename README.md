# Cart-IU - Loja Virtual com Carrinho de Compras

Uma **loja virtual moderna** com catálogo de produtos, carrinho completo e interface intuitiva. Desenvolvido para demonstrar habilidades em JavaScript, manipulação do DOM, consumo de API e CSS avançado.

**[Ver demonstração ao vivo →](https://vitor213.github.io/Cart-IU/)**

## ✨ Funcionalidades

- **Catálogo dinâmico** com produtos da [FakeStore API](https://fakestoreapi.com/)
- **Busca em tempo real** com debounce
- **Filtros** por preço e ordenação
- **Carrinho completo** (adicionar, aumentar/diminuir quantidade, remover)
- **Animação fly-to-cart** ao adicionar produto
- **Modal** de detalhes do produto
- **Toast notifications**
- **Persistência** dos itens no `localStorage`
- **Design responsivo** e moderno
- **Loader** e tratamento de estados vazios

## 🛠️ Tecnologias Utilizadas

- **HTML5** Semântico
- **CSS3** (Flexbox, Grid, Variáveis, Animações, Gradientes)
- **JavaScript Vanilla** (ES6+)
- Consumo de API REST
- LocalStorage
- Mobile-first
